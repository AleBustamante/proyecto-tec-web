Los lenguajes de programación han evolucionado significativamente desde los lenguajes de bajo nivel como el ensamblador, hasta lenguajes modernos de alto nivel como Python y Java.

Los lenguajes de bajo nivel están más cerca del hardware, mientras que los lenguajes de alto nivel permiten el desarrollo más rápido y la escritura de código más legible.

Lenguajes como C y C++ se utilizan en sistemas embebidos,
mientras que lenguajes como Python son preferidos para desarrollo web.

/usr/local/bin/python3
